# 🧮 برنامج "يَقين" المحاسبي الذكي

برنامج محاسبي متكامل يدعم متطلبات الزكاة والضريبة السعودية.

## 🚀 البدء السريع

1. تثبيت المكتبات:
\`\`\`bash
cd frontend
npm install
\`\`\`

2. تشغيل الخادم:
\`\`\`bash
npm start
\`\`\`

3. فتح المتصفح:
\`http://localhost:3000\`

## 📁 هيكل المشروع

- \`frontend/\` - واجهة المستخدم (React)
- \`backend/\` - الخادم والواجهة البرمجية
- \`database/\` - مخطط قاعدة البيانات
- \`documentation/\` - التوثيق
